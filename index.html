📖 深度使用指南與設計邏輯

本工具不僅是測驗機，更是基於認知心理學設計的記憶強化系統。以下是各項核心功能的詳細解析：

1. 結構化題庫管理 (Structured Organization)

具體操作：在「題庫管理」中建立不同名稱的資料夾（如：TOEIC 單字、課本 L1）。

設計初衷：人類大腦對於有規律、有分類的資訊吸收最快。透過分類，您可以縮小單次學習的範疇，建立大腦的「索引標籤」，避免資訊過載產生的混淆。

2. 智慧化錄入系統 (Smart Entry)

具體操作：單筆新增適合零星補充；「🚀 批次匯入」則支援 英文 中文 格式的文字直接貼上。

設計初衷：錄入資料不應成為學習的阻礙。批次功能是為了讓您將外部教材（如 PDF、網頁單字表）快速轉化為數位練習題，將寶貴的精力集中在「記憶」而非「打字」。

3. 階梯式測驗模式 (Scaffolded Testing)

具體操作：先進行「英翻中選擇」熟悉單字形象，再進行「中翻英填充」挑戰拼寫。

設計初衷：遵循「由淺入深」的學習曲線。選擇題練習的是「再認記憶」（Recognition），而填充題則是難度最高的「回憶記憶」（Recall）。兩者並行能確保從看到單字有印象，進步到能完整運用。

4. 答對率排序邏輯 (Data-Driven Priority)

具體操作：編輯列表會動態將答對率最低的單字置頂，並以藍色與綠色方塊標示熟練度。

設計初衷：落實「精準練習」。大多數人傾向重複練習已經會的單字，這會造成效率低下。系統強制將您的「弱點」推至眼前，讓每一分鐘的複習都發揮最大價值。

5. 自定義難度與鷹架提示 (Adaptive Difficulty)

具體操作：測驗前可設定「計時秒數」增加難度，或勾選「首尾提示」降低挫折感。

設計初衷：計時器是為了訓練「自動化反應」（不經思考就能反射單字意），模擬真實對話壓力。首尾提示則是學習心理學中的「鷹架功能」，在您差一點就想起來時給予關鍵助推。

6. 即時回饋與閉環複習 (Feedback Loop)

具體操作：答錯時立即彈出答案，測驗結束後提供「一鍵複製錯題」。

設計初衷：大腦在犯錯的那一刻學習力最高。即時糾正能中斷錯誤的聯結；複製錯題功能則讓您可以將這份清單再次匯入新題庫，形成「練習→發現錯誤→專項複習」的完美閉環。

7. JSON 本地備份機制 (Data Portability)

具體操作：定期使用「匯出備份」下載檔案，或在換手機時使用「匯入」讀取舊資料。

設計初衷：為了隱私，您的數據僅儲存在您自己的手機裡。但網頁緩存可能遺失，定期匯出 JSON 不僅是為了安全，更是讓您可以將自己辛苦累積的數位資產永久保存。
